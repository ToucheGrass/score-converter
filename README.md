# Score Converter

Convert your CloverPit's score into a valid time for Speedrun.com leaderboard.

## Content:

- `scoreconverter.exe` : main .exe

## Usage

Double-click son the `.exe` to launch the app.
Enter your score in the box in this exact format : "x.xxxxxE+xxx". The first digit will determine the number of minutes,
both 2nd and 3rd determine the number of seconds, both 4th and 5th digits determine the milliseconds and the last 3 digits determine the number of hours.
